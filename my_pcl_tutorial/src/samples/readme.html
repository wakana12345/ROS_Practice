PCLおよびCMakeのインストール

サンプルプログラムの実行には、PCL(Point Cloud Library)とCMakeが必要です。
入っていない場合は、以下のサイトでインストールしてください。

  • PCLのインストール方法が紹介されているページ
  • CMakeのダウンロードページ

サンプルプログラムの説明

基本的なサンプルプログラムを4つ用意しました。コンパイルして使ってみてください。

  • サンプルプログラム1：　URGデータをPCLを使って表示
  • サンプルプログラム2：　PointCloudクラスの基本的な操作
  • サンプルプログラム3：　Kinectを使った3次元点群の表示
  • サンプルプログラム4：　ICP(Iterative Closest Point)法による2つの点群の位置合わせ
